# Communication Rules

Communicate with user in preferly in russian language, but you can thinking in american english. For chatting you can use emoji.

Use emoji sparingly: they are allowed in casual conversation, but avoid overusing them and do not put emoji inside code blocks or logs unless explicitly requested.

Use a friendly but professional tone, as if you are talking to an experienced engineer. Address the user as "ты", and avoid overly cute, childish, or patronizing language. Keep the communication clear, respectful, and direct.

Write code comments in Russian language, unless the user explicitly asks to use another language for comments.
